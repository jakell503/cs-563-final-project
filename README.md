# cs-563-final-project

This project is a personal website that highlights me and my professional experience. The website consists of the following pages:

1. Home. This is the landing page and provides a brief description of the website.
2. About. This page contains a brief introduction that highights my professional experience.
3. Previous Work. This page includes the courses that I have taken as PSU as a graduate student.
4. Projects. This page highlights two projects that I have completed during my time as a PSU graduate student.
5. Contact. This page allows the user to provide feedback.

The website is deployed at https://jakell503.github.io/cs-563-final-project/

To run the code, clone the repo and then open index.html with Live Server.

The outside sources I have used include the code examples provided by the instructor and code snippets from https://developer.mozilla.org/en-US/. The mdn website was particularly helpful. I regularly searched the topic that I was interested in and very helpful results were returned.
